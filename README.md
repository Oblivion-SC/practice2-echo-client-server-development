## Описание проекта
Данный проект реализует простой TCP эхо-сервер и клиент на языке Python с использованием низкоуровневого механизма сетевого взаимодействия Berkeley Sockets. Сервер принимает текстовые сообщения от клиента, обрабатывает их (переводит в верхний регистр) и возвращает обратно.

---
## Цель работы
Изучить низкоуровневый механизм сетевого взаимодействия (Berkeley Sockets) и реализовать протокол передачи данных по TCP на языке Python.

## Инструкция по запуску
**Шаг 1: Проверка установленной версии Python**

**Откройте терминал (командную строку) и выполните:**
```
python --version
```
**Шаг 2: Запуск сервера**
1. Откройте первый терминал\
2. Перейдите в папку с проектом\
3. Запустите сервер:
```
python Server.py
```
4. Сервер перейдёт в режим ожидания подключения:
```
[СЕРВЕР] Запущен на 127.0.0.1:12345
[СЕРВЕР] Ожидание подключений...
```
**Шаг 3: Запуск клиента**
1. Откройте второй терминал
2. Перейдите в папку с проектом
3. Запустите клиента:
```
python Client.py
```
4. Клиент подключится к серверу:
```
Попытка подключения к 127.0.0.1 : 12345...
Подключение успешно установлено.
```
**Шаг 4: Обмен сообщениями**
1. В окне клиента вводите текстовые сообщения
2. Наблюдайте за логами в обоих окнах
3. Сервер возвращает сообщения в верхнем регистре

**Пример:**
```
Введите сообщения. Для выхода введите 'exit' или 'quit'
Ответ сервера: 'ECHO: ПРИВЕТ, КАК ДЕЛА?'
```
**Шаг 5: Завершение работы**

**Для корректного завершения сеанса введите команду:**
```
exit
```
**Оба приложения закроют соединения и завершат работу.**
